== Description ==

Display data avaiable from the [AddThis.com Analytics API](http://www.addthis.com/help/analytics-api) in the sidebar of your site.

Multiple widgets can displayed on a page with customisable titles.

There is a widget interface to allow you to choose which data to share.  The data is output in a table with
CSS classes to hook styling to your data.

Future improvements will include the introduction of integration with [Google Charts API](http://code.google.com/apis/chart/) or a similar charting library.

== Installation ==

Drop addthis folder into /wp-content/plugins/ and activate the plug in the Wordpress admin area.
